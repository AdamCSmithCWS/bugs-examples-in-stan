General Comments
November 10, 2005: My experience is that the examples using logit and/or item-response models (e.g., Turnout, Legislators, Judges) will generate errors with an out-of-the-box version of WinBUGS or OpenBUGS. I force the software to use slice sampling instead of the default derivative-free adpative rejection sampling algorithm, as follows:
  • WinBUGS: See the WinBUGS documentation on changing defaults for sampling methods.
  • OpenBUGS: You can disable rejection sampling permanently either by (a) editing Updater/Rsrc/Methods.txt (although I've never done that in OpenBUGS, and in OpenBUGS this file has a different layout to its WinBUGS counterpart), or (b) using the Options menu to disable the Rejection class of updating methods.
