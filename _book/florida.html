Bayesian Model Examples

3 Florida: Learning About an Unknown Proportion from Survey Data

library("rstan")

This is an learning about an unknown proportion from survey data; using survey data to update beliefs about support for Bush in Florida in the 2000 presidential election campaign (Jackman 2004).

florida_mod <- stan_model("stan/florida.stan")
  data {
  real y;
  real y_sd;
  real mu_mean;
  real mu_sd;
}
parameters {
  real mu;
}
model {
  mu ~ normal(mu_mean, mu_sd);
  y ~ normal(mu, y_sd);
}

The prior polls had a mean of 49.1% in support for Bush, with a standard deviation of 2.2%. The new poll shows 55% support for Bush, with a standard deviation of 2.2%.

florida_data <- list(
  mu_mean = 49.1,
  mu_sd = 2.2,
  y_sd = 2.2,
  y = 55  
)
florida_fit <- sampling(florida_mod, data = florida_data)
florida_fit

After observing the new poll, the mean for the posterior is 52, with a 95% credible interval of 49–55.
