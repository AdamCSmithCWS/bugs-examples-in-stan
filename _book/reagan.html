7 Reagan: linear regression with AR(1) disturbances

library("tidyverse")
library("rstan")

Ninety-six monthly observations on presidential job approval ratings for Ronald Reagan are modeled via linear regression, with a correction for first-order serial correlation among the disturbances.5 Note the marginal model for the first observation, and the conditioning on the lagged observation for months 2 through 96. A uniform prior over the stationary (-1,1) interval is employed for the residual AR(1) parameter.

\[ \begin{aligned}[t] y_i &= \mu_i + \epsilon_i + \theta \epsilon_{i - 1} ,\\ \mu_i &= \alpha + x_i' \beta , \\ \epsilon_i &\sim \mathsf{Normal}(0, \sigma^2) , \end{aligned} \] for \(i \in 1, \dots, N\). Weakly informative priors for each parameter are used, \[ \begin{aligned}[t] \alpha &\sim \mathsf{Normal}(0, 10), \\ \beta_k &\sim \mathsf{Normal}(0, 2.5), & k \in 1, \dots, K, \\ \sigma &\sim \mathsf{HalfCauchy}(0, 5), \\ \theta &= 2 \theta^* - 1 , \\ \theta^* &\sim \mathsf{Beta}(1, 1) . \end{aligned} \]

  alpha_loc <- 0
  alpha_scale <- 10
  beta_loc <- rep(0, K)
  beta_scale <- rep(2.5 * sd(y), K)
  sigma_scale <- 5 * sd(y)
  theta_a <- 1
  theta_b <- 1
})

mod_regar1
  data {
  // number of observations
  // need at least two to estimates
  int N;
  // response
  vector[N] y;
  // regression design matrix
  int K;
  matrix[N, K] X;
  // priors
  real alpha_loc;
  real alpha_scale;
  vector[K] beta_loc;
  vector[K] beta_scale;
  real sigma_scale;
  real theta_a;
  real theta_b;
}
parameters {
  // regression coefficients
  real alpha;
  vector[K] beta;
  // error scale
  real sigma;
  // lag coefficients
  real theta_raw;
}
transformed parameters {
  // observation means
  vector[N] mu;
  // error terms
  vector[N] epsilon;
  // lag coefficient;
  real theta;
  // convert range of theta from (0, 1) to (-1, 1)
  theta = (2. * theta_raw - 1.);
  // regression
  mu = alpha + X * beta;
  // construct errors
  epsilon[1] = y[1] - mu[1];
  for (i in 2:N) {
    epsilon[i] = y[i] - mu[i] - theta * epsilon[i - 1];
  }
}
model {
  alpha ~ cauchy(alpha_loc, alpha_scale);
  beta ~ cauchy(beta_loc, beta_scale);
  theta_raw ~ beta(theta_a, theta_b);
  sigma ~ cauchy(0, sigma_scale);
  for (i in 2:N) {
    y[i] ~ normal(mu[i] + theta * epsilon[i - 1], sigma);
  }
}

7.1 Cochrane-Orcutt/Prais-Winsten

An AR(1) error model can also be estimated Prais-Winsten estimation: \[ \begin{aligned}[t] y_1 &\sim \mathsf{Normal}\left(\alpha + x_1' \beta, \frac{\sigma ^ 2}{1 - \theta ^ 2} \right), \\ y_i &\sim \mathsf{Normal}\left(\theta y_{i - 1} + \alpha (1 - \theta) + \beta (X_i - \theta X_{i - 1}), \sigma ^ 2 \right) & i = 2, \dots, N \end{aligned} \]

mod_pw
  data {
  // number of observations
  // need at least two to estimates
  int N;
  // response
  vector[N] y;
  // regression design matrix
  int K;
  matrix[N, K] X;
  // priors
  real alpha_loc;
  real alpha_scale;
  vector[K] beta_loc;
  vector[K] beta_scale;
  real sigma_scale;
  real theta_a;
  real theta_b;
}
parameters {
  // regression coefficients
  real alpha;
  vector[K] beta;
  // error scale
  real sigma;
  // lag coefficients
  real theta_raw;
}
transformed parameters {
  // observation means
  vector[N] mu;
  // lag coefficient;
  real theta;
  // convert range of theta from (0, 1) to (-1, 1)
  theta = (2. * theta_raw - 1.);
  // regression
  mu[1] = alpha + dot_product(beta, X[1, ]);
  mu[2:N] = alpha * (1 - theta) + (X[2:N, ] - theta * X[1:(N - 1), ]) * beta;
}
model {
  alpha ~ cauchy(alpha_loc, alpha_scale);
  beta ~ cauchy(beta_loc, beta_scale);
  theta_raw ~ beta(theta_a, theta_b);
  sigma ~ cauchy(0, sigma_scale);
  y[1] ~ normal(mu[1], sigma / sqrt(1 + theta ^ 2));
  y[2:N] ~ normal(mu[2:N], sigma);
}
